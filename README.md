# Hi, I'm Alastair 👋

BI & Analytics specialist with 15+ years' experience across Power BI, T-SQL, DAX and Excel-native BI builds — semantic modelling, commercial reporting and customer analytics for retail, gaming and automotive.

This repo holds worked examples pulled from real production models and reports (anonymised or redacted where the underlying data is commercially sensitive). Each folder below has its own `Overview` file with more detail.

## What's here

| Folder | Demonstrates |
|---|---|
| **DAX Examples** | Full DAX measure catalogues from two production semantic models — table functions, time intelligence, a Calculation Group (`SELECTEDMEASURE()`-driven R6M/MTD/QTD/YTD), `USERELATIONSHIP`-based multi-relationship handling, and a native DAX implementation of market-basket analysis (support / confidence / lift) |
| **T-SQL Use Cases & Examples** | Staged temp-table pipelines, multi-year fact-table unions, geospatial catchment analysis (`STDistance` against geography types), and GDPR suppression logic embedded directly in customer-facing queries |
| **M-Code Examples** | Power Query patterns — parameterised and rolling date tables, and native SQL query folding inside M |
| **Customer Segmentation (T-SQL & Excel)** | RFM-style frequency/value clustering, plus an automated insight-narrative generator built from nested Excel formula logic — translating segment index values into plain-English callouts without manual write-up |
